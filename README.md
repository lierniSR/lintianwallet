# Lintian Wallet
Hoy en día, muchas personas tienen dificultades para llevar el control de sus gastos, por esta razón, el año pasado Banco Hispania nos pidió crear una aplicación móvil, pero a la vista de que muchos usuarios se han quejado por tema de espacio en sus teléfonos móviles, este año han decidido contar con nuestros servicios de nuevo para duplicar la aplicación móvil en una página web, de esta manera no tendrán que preocuparse del espacio de sus teléfonos. La aplicación puede registrar gastos, visualizar el estado de sus gastos personales y establecer presupuestos para mejorar el ahorro. Con Lintian Wallet, Banco Hispania busca ofrecer a sus clientes otra herramienta digital que les ayude a organizar mejor sus gastos personales y tomar mejores decisiones a la hora de ahorrar.
